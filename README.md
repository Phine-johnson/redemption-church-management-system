# Church Management System

This project is split into:

- `client`: React frontend
- `server`: Node.js and Express backend

## Quick start

1. Install dependencies:
   `npm install`
2. Start the frontend:
   `npm run dev:client`
3. Start the backend:
   `npm run dev:server`
